Getting Started

I recommend that you try out the following steps, it should work:

  1. Click Reset button, it will load the default test event and show the race table below the input area.
  2. The test event has two races and one intermediate time point.
  3. Navigate through the races with R- and R+ buttons, and through the time points with IT- and IT+
  4. Note that there is a time point called IT0, which is the finish of the race.
  5. Click Next button, it will take you back to Race 1 - IT 1, the first mark rounding in the race.
  6. Because button Next has figured out where we are in the progress of the race - at the beginning.
  7. Notice how the current race and current IT are displayed as part of the input header line.
  8. Click on the circular fab buttons (floating action buttons).
  9. At least once: select the dnf radio button before you click.
  10. Notice how the radio selection will jump back to time when you click a fab.
  11. The fab just clicked will turn blue.
  12. Click Next again when all fabs have been used up.
  13. You will be advanced to the finish time point of race 1, please check.
  14. Click Rows and Event to show the event table in addition to the race table.
  15. Continue with the fabs and this time watch both tables.
  16. Play with the toolbar icon buttons, most of them have the Reduce To action type.
  17. Some of the buttons will toggle, some will show, hide, or have an action type of Reduce To.
  18. Go ahead and input timing info for Race 2.
  19. You can skip IT1 and only do the finish.
  20. There are only two races in this test event.
  21. When done, examine that series scoring is correct in table event.
  22. Adjust the number of throwouts with button T+ and T-.
  23. Note that you can sort the html table with a click on the column header.
  24. Find the Pre-Text area and use it to copy result data in compact format to the clipboard.
  25. Try and paste the text data you just copied into your favorite code editor program.
  26. I assume that you test in Chrome on the desktop before you use it on a tablet or even a phone.

You need to input a finish position or a status value for all the entries in a race.

I will go through some terms quickly.

  • A regatta has events.
  • An event has races.
  • A race has time points.
  • At minimum, there is the finish time point.
  • You may set up some intermediate time points.
  • Remember that IT0 is the finish time point.
  • IT1 is the first time point and so on.
  • You need to do the finish; all other time points are optional.
  • The entries in a race are called bibs, corresponding to bows or bow numbers.
  • By tapping on fabs you finish off bibs until done.
  • You can assign a status value like dnf to a bib instead of a time.
  • You can also erase the time for the current bib or reset the status to ok.
  • A bib with a status other than ok will not have a fab on the next IT.
  • A bib with a finish position other than 0 for the race in the event will also be excluded.
